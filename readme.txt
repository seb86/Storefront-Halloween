== Description ==

A Halloween child-theme for Storefront that makes your site go bump in the night.

Use Font Family: Nunito
The Halloween theme style runs between 28th October to 2nd November then reverts to Storefront default while keeping some of the tweaks made.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Storefront Halloween in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
